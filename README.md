# Feirm Recovery

This website will allow you to recover your private keys for various cryptocurrencies listed on the [Feirm Platform](https://app.feirm.com) into the WIF format. If needed, this means you can import your keys into official QT wallet created by those projects.

## Building from scratch
```bash
$ git clone https://github.com/jackcoble/feirm-recovery.git
$ cd feirm-recovery/
$ yarn install
$ yarn serve
```
